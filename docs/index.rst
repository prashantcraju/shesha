Shesha Documentation
====================

**Shesha** is a Python library for measuring the geometric stability of high-dimensional
representations via self-consistency of Representational Dissimilarity Matrices (RDMs).

.. image:: https://img.shields.io/pypi/v/shesha-geometry.svg
   :target: https://pypi.org/project/shesha-geometry/
   :alt: PyPI version

.. code-block:: bash

   pip install shesha-geometry
